How to get a token

Go to GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token. Give it access to only this repo, with Repository permissions → Contents: Read and write. Paste it above. It's stored in your browser's localStorage and never leaves this page except to call api.github.com.
